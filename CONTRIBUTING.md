# Contributing

Thanks for your interest in WayFinder.

This is a public showcase repository, so contributions should stay within the safe public scope.

## Good Contributions

- Documentation improvements
- UI polish
- Accessibility fixes
- Public demo data improvements
- Tests for public demo logic
- Issue template improvements

## Avoid

- Adding secrets or `.env` files
- Adding database dumps or generated local data
- Adding proprietary datasets
- Copying private production ranking logic
- Uploading research PDFs or internal notes

## Workflow

1. Open an issue describing the change.
2. Create a branch with a clear name.
3. Keep the pull request focused.
4. Include screenshots for UI changes.
5. Explain any architecture changes.
