# Public Repository Scope

This repository is a curated public showcase.

## Included

- Product-level frontend demo
- Public-safe backend API shell
- Simplified AI engine modules
- Architecture documentation
- Contribution and GitHub workflow templates
- Placeholder asset structure

## Excluded

- Production `.env` files
- API keys and credentials
- MongoDB binaries or local data
- Database dumps, lock files, and generated state
- Raw proprietary place intelligence datasets
- Advanced optimization experiments
- Monetizable internal ranking heuristics
- Private prompts or vendor orchestration logic
- Research PDFs and temporary documents

## Intent

The repo should be useful for recruiters, collaborators, and technical reviewers while remaining safe to publish.
