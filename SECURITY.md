# Security

Do not open public issues containing secrets, credentials, private datasets, or user data.

If you find a security concern in this showcase repository, please contact the maintainer privately through the GitHub profile.

## Public Repo Safety Rules

- Do not commit `.env` files.
- Do not commit MongoDB files, dumps, or local runtime artifacts.
- Do not commit production prompts, API keys, or vendor configuration.
- Do not commit real user data.
